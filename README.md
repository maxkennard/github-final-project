# Simple Interest Calculator

A simple Python program that calculates simple interest based on user-provided principal amount, rate of interest, and time period.

## Formula

Simple Interest (SI) is calculated using the formula:

    SI = (P × R × T) / 100

Where:
- **P** = Principal amount
- **R** = Rate of interest (per annum)
- **T** = Time period (in years)
